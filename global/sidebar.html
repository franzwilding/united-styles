Fixable and Direction

Sidebars are fixeable, when you add the class .fixed and a direction (.top, .right, .bottom, .left), the sidebar will be fixed to the side of the window.

On small devices, the sidebar will be hidden by default and shown 100% width, when a class .open is added.

A fixed sidebar should always be placed before a content element in the DOM, because it will add margin to the next element.
